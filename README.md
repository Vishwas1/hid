# HID

- Solidity version: `0.6.2`
- Node version: `v13.12.0`
- Truffle Version: `v5.2.0`

## Run local blockchain

```
ganache-cli -m <mnemonics>
```

## setup

```
npm i
```

## compile
```
truffle compile
```

## Deploy

```
truffle migrate --network development
```

## Test

```
truffle test
```
